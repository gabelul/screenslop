# Research Workspace

Screenslop has an ignored local research area for inspiration material, cloned repos, and copied agent skills.

```text
research/repos/     # cloned public repos to inspect locally
research/skills/    # copied skill docs from Codex, Claude, local agents, etc.
research/findings/  # notes from what looked useful
```

These folders are intentionally untracked. They are for learning, not shipping.

Good candidates:

- Impeccable, for command router and taste-system ideas.
- Pixelslop, for evidence-first browser review workflow.
- Baguette, for iOS simulator runtime control.
- XcodeBuildMCP, for agent-native Xcode fallback workflows.
- Local SwiftUI skills from Codex/OpenAI curated skill packs.

When something graduates from research into Screenslop, write a short note in `docs/` explaining what was adopted and why. Do not leave important decisions trapped in ignored folders.
